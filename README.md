# Todo List

## System Requirements
- PHP **V8**
- Composer **^V2**

## Installation

```
git clone https://github.com/ezehkingsleyuchenna/todolist.git
```
```
composer install
```
```
npm install
```

**Create database.sqlite in database directory**

```
php artisan migrate
```

## Running Application

```
npm run dev
```
