# Todo List

## System Requirements
- PHP `v8.1`
- Composer `^v2`

## Installation

```
git clone https://github.com/ezehkingsleyuchenna/todolist.git
composer install
```

### Environment file

- **Create `.env` file in the root folder**

- **Copy `.env` data from `.env.example`**

### Generate App key

```
php artisan key:generate
```

### Database
**Create `database.sqlite` in database directory**

### Migrate tables
```
php artisan migrate
```

### NPM
```
npm install
```

### Running Application

```
npm run dev
```
